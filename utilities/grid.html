Basic Grid

The basic grid is initiated by adding the .fl class to an element. By default, display: flex; starts a non-wrapping row. To convert it to a column, add the .fl-col class.

Grid Cells

Fluid Cells

Add the .-cell class to any number of items within a container and let the grid figure it out. The .-cell class has a flex: 1 0 auto; value by default. This means the cell's width will be automatically determined by the cell's content.

But sometimes you want to be able to have all cell's within a box container to be the same width. Instead stating a fixed cell width, apply a modifying to the box container and all child elements will automatically be sized.

Grid Spacing

Grids, by default, have no spacing between cells. However sometimes spacing is desired between cells. This spacing though needs to our standard spacing units: 2px, 4px, 8px, 12px, 16px, 24px, 32px, 48px, and 64px. Grid spacing can be applied with a ._sp-[X] modification class. Note: the spacing unit is the total space between each cell, not the space added to each cell.

Cell Width: Flex Grow

These modifying classes alter the flex-grow property. The default value is 1.

On parent containers, you can add .fl0, which will apply to all child cells within it. You can also modify individual cells with the following classes: .fl0, .fl1, .fl2, .fl3, .fl4, and .fl5. What a non-zero number means is you are telling flexbox how large that cell should be in relation to all other cells within that parent.
