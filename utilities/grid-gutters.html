Gutters

Grids, by default, have no spacing between cells. However sometimes spacing is desired between cells. This spacing though needs to our standard spacing units: 2px, 4px, 8px, 12px, 16px, 24px, 32px, 48px, and 64px. Grid spacing can be applied with a ._sp-[X] modification class. Note: the spacing unit is the total space between each cell, not the space added to each cell.
