Cell Width: Flex Grow

These modifying classes alter the flex-grow property. The default value is 1.

On parent containers, you can add .fl-0, which will apply to all child cells within it. You can also modify individual cells with the following classes: .fl-0, .fl-1, .fl-2, .fl-3, .fl-4, and .fl-5. What a non-zero number means is you are telling flexbox how large that cell should be in relation to all other cells within that parent.
