Labels

Labels are used to inform users what information is being asked of them. Labels should be written in sentence case. For usability reasons, if a label is connected with an input, the for="…" attribute should be filled in. The for="…" attribute references the input's id="…" value. Additionally if you would like to add required or optional labels, whenever possible use the full word label instead of asterisks, as this makes it more flexible for translations.

Sizes & Styles

Labels are available in four sizes, which are outlined within the input sizes area. Besides sizes, you can also adjust a label's font-weight by applying a ._fw-400 modifying class.
