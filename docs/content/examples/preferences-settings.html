{% header "h2", "Write labels in the affirmative" %}

Preferences and settings should always be written in the affirmative. For example, your preference should be written as “Enable dark mode.” Even if the preference is meant to disable a feature, it should still be written in the affirmative.

When a user checks a feature preference, they should be getting that feature.
