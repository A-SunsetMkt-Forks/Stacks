Note: The unset property is a combination property that resets a property to its inherited value. Meaning it behaves like the inherit and the initial keywords, but it properly selects the right value for you. Unfortunately it is not fully supported in IE yet. Because of this all unset values have had the correct inherit or initial values applied.
