{% header {element: 'h2', text: 'Base Style'} %}

The progress bars use background gradients for the fill color. In order to properly reveal the changing color of the bar, the progress "fill" is the actual background gradient. The actual bar is a gray mask that slowly shrinks to the right to reveal the background gradient underneath.
