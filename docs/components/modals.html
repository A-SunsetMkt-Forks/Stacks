Confirmation

Concise

This modal is used to confirm a user’s action. It is intended to add friction to an interaction, only confirming the action if the user explicitly presses "confirm". In this example, we present the modal to confirm they’re posting a question on public Stack Overflow.

{% endhighlight %}
Are you sure you want to ask this question publicly?
