General Usage

Modal windows intended to quickly grab the user's attention. Complicated interfaces or decisions are better served by their own page.

Behavior

Since Stacks’s scope is limited to CSS and HTML, the JS found on these pages is intended to communicate behavior only.

Modals should be presented with a semi-transparent blocker to focus the user’s attention on the task being presented. Practically, on Stack Overflow, most pages will have a div with the class .s-modal-overlay somewhere in the view. You’ll need to include one if it’s not. The modal experience is not complete without the overlay. In addition, our modals are not feature-complete unless the following behaviors are enabled:

  1. The user can close the modal dialog with their ESC key.
  2. The user can click outside the dialog to close it.
