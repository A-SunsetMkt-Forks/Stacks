--- layout: page title: Filters description: Filters are a horizontal collection of buttons used to filter a list or table. If the content you’re interacting with is a simple paring down of the current view, it’s appropriate to use the .s-filters component. Add the class .is-selected to show the currently-enabled filter. This component is used in our questions view, and is frequently used in conjunction with other form elements such as search fields and sorting dropdowns. ---
