Notices are offered in three different visual styles:

  • Outlined: Minimal style used to minimally separate notices from the main content
  • Filled, Muted (Default): Default style used to separate notices from the main content
  • Filled, Important: Used sparingly for when an important notice needs to be noticed

Notices can be deployed in four different ways:

  • Inline (Default): Default behavior for notices that are inserted within the content area
  • Floating: Floats a notice above the current page's content
  • Banner: Full width notices usually at the top or bottom of the browser window
  • Toast: Floating notices, but aligned to the top and center of the page and they disappear after a set time
Class Applied to Description
.s-notice N/A Parent notification style
.s-notice__outlined .s-notice Changes the visual style to a minimal, outlined notice
.s-notice__important .s-notice Changes the visual style to a heavy, loud important notice
.s-notice__info .s-notice Visual style for information notices
.s-notice__success .s-notice Visual style for success notices
.s-notice__warning .s-notice Visual style for warning notices
.s-notice__danger .s-notice Visual style for danger notices
.s-notice__float .s-notice Changes the notice behavior from an inline content to floating above content
.s-notice__banner .s-notice Changes the notice behavior from an inline content box to a full-width site banner
.s-notice__toast .s-notice Changes the notice behavior to a floating notice aligned to the top, center of the website. This disappears after a set time.
