Responsive utility classes?

Many utility classes in Stacks are also available in screen-size specific variations. For example, the .d-none utility can be used on small screen sizes and below using the .sm:d-none class, on medium screen sizes and below using the .md:d-none class, and on large screen sizes and below using the .lg:d-none class.

This is done using predefined max-width media query breakpoints represented by t-shirt sizes. Any classes you apply at desktop screen sizes are also applied to smaller sizes. A common example would be to apply .md:d-none to an illustration. This means, “At the medium breakpoint and smaller, apply d-none.”

Responsive utility availability

A limited selection of classes are available as breakpoint-based overrides. These are represented by [ICON].
