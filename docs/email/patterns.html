{% header {element: 'h2', text: 'Team Identification'} %}

Emails coming from a private Team should be clearly labeled by placing a Team Icon and Name pattern in a prominent place. An icon-only version is also available if spacing is tight.

An email should only contain content from one Team. A Team's content should never appear mixed alongside with content from another Team or public content in a single email.
