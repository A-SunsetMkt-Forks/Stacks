{% header {element: 'h2', text: 'Install a full Ruby development environment'} %}

You'll need a full Ruby development environment in order to run our static site generator.

  1. Install the command line tools `xcode-select --install`
  2. Accept the Xcode license. `sudo xcodebuild -license accept`
  3. Install Homebrew to install a fresh updated version of Ruby `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`
  4. Install a fresh version of Ruby `brew install ruby`
  5. Quit your Terminal and relaunch it. Don't just close the window. Quit the whole app.
