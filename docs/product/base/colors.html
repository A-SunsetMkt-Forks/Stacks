{% header "h2", "Stops" %}

Our colors are each available in a number of stops from dark to light—900 being the darkest, and 025 or 050 being the lightest. All stops are available as CSS and Less variables. Hex values are provided for portability outside of Stacks only, and should not be directly referenced in CSS.

Display

{{ color.name | capitalize }}
{% for stop in color.stops reversed %}
var(--{{ color.name }}-{{ stop.stop }}) @{{ color.name }}-{{ stop.stop }} #{{ stop.hex }}
{% endfor %}
{% endfor %}
{% header "h2", "Classes" %}

Each color stop is available as an atomic text, background, and border class. Using these atomic classes means your view will respond to dark mode appropriately. These colors are available conditionally.
