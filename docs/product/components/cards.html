{% header h2 | Base %}

The base card styling applies a border to the card. Cards should use a heading that sets clear expectations about the card’s purpose, paragraphs that put the most critical information first, and (optionally) calls to action on the bottom to direct user action.

{% header h2 | Linked Cards %}

The .s-card class can be applied to an <a> tag for instances where a whole card should link somewhere.

A :hover style is automatically added to all cards. For linked cards with a .bs-* adding a .bs-hover class will apply a hover style to the box shadow as well.
