{% header h3 | Basics %}

From a purely visual standpoint, all you need is a container of class s-collapsible and a child of class s-collapsible--content. Toggle the state class is-collapsed on the container to collapse / expand the content. Care must be taken to keep the collapsing functionality accessible.

{% header h3 | JavaScript %}

In the usual case in which you have a button or other clickable element that toggles the collapsed state, you can use the built-in s-collapsible-control functionality. See the JavaScript introduction for general information on JS in Stacks.

  1. Give the s-collapsible an id attribute.
  2. Give your button an aria-controls attribute whose value is the ID from 1.
  3. Give your button an aria-expanded attribute whose value is true or false depending on the initial state of your collapsible.
  4. Give your button a data-controller="s-collapsible-control" attribute.
  5. If your button is not an actual button or anchor element, be sure to make it keyboard-reachable with tabindex="0"
  6. Give your button an aria-label attribute that makes it clear that this is a collapsing control, unless this is already clear from its text content
