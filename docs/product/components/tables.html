{% header {element: 'h3', text: 'Text Alignment'} %}

Aligns text horizontally, at either the table-level or cell-level. In either case, using atomic classes assigns how cell(s) should horizontally align text: .ta-left, .ta-center, .ta-right, or .ta-justify.

Columns should usually be left-aligned unless the content within the column is a number that expresses monetary value, is a value, or is a percentage. Those should be right-aligned. Number displays that are always fixed (such as phone numbers) should be left-aligned.
