{% header h2 | Example %}

Breadcrumbs show users where they are in the website hierarchy and how to navigate to previous page levels. It can be appropriate to use a breadcrumb when:

  • The user is many levels deep
  • The current page does not have its own navigation
  • The user needs to quickly go back to the previous parent page
