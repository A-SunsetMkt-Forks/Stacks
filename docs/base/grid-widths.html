Grid Cells

Fluid Cells

All flex cells inherit the browser flex default: flex: 1 0 auto; This means that by default all cells will be as wide as their content. If you would like a cell or all the cells to fill the remaining space, apply the .fl1 to the cell or .grid__fl1 to the parent.

Auto Fixed Width Cells

Automatic adjusting width cells are great. And being able to specify a cell’s width is equally amazing. But sometimes you want to be able to have all cell’s within a box container to be the same width. Instead stating a fixed cell width, apply a modifying to the box container and all child elements will automatically be sized.
