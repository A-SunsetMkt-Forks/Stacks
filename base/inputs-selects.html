Selects

Select menus are UI elements which present a list of options for users. Select menus should be used when you have more than 3 options. If you have 3 or less options, consider using a toggle instead.

Select Style

To get the customized select menus, wrap the select tag with a wrapping .select div. This clears out the browser default styling for select menus and re-styles the menu.
