Gutters

Grids, by default, have no gutters between cells. When gutters are desired between cells, you apply them to the parent .grid class. The gutter applies a margin to all sides. The available gutter sizes are 2px, 4px, 8px, 12px, 16px, 24px, 32px, 48px, and 64px). Please note that the number in the grid spacing class name (i.e. gs4) is the desired total gutter space between each cell.

Row or Column Gutters Only

Flexbox doesn't collapse margins like other CSS elements. Since margins compound, sometimes there's a need to remove gutters from the top and bottom (column) or left and right (row). To do this, add the following modifying classes. To have gutters only in a row, apply .gsx. To have gutters only in a column, apply .gsy.
