Gutters

Grids, by default, have no gutters between cells. When gutters are desired between cells, you can apply to them universally to grid items, along the x-axis, or the y-axis. Our gutters follow our spacing system values (2px, 4px, 8px, 12px, 16px, 24px, 32px, 48px, and 64px).

Gutters can be applied universally (all sides, .gs[#]), only on the x-axis (.gsx[#]), or only on the y-axis (.gsy[#]). Please note that the number in the spacing unit class name (i.e. gs4) is the desired total gutter space between each cell.
